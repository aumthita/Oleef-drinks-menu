
One-page Drink Menu Site (TH)
=============================

โฟลเดอร์นี้คือเว็บหน้าเดียวที่แสดงรูปเมนูของคุณเท่านั้น
ใช้งานแบบง่ายสุด 3 ขั้นตอน:

1) เปลี่ยนรูป
   - ลบไฟล์ menu.jpg แล้วใส่รูปเมนูของคุณเป็นชื่อ "menu.jpg"
   - แนะนำขนาดด้านกว้าง ~1080-1600px, ไฟล์ < 1MB เพื่อโหลดไว

2) ทดสอบบนเครื่อง
   - ดับเบิลคลิก index.html เพื่อเปิดบนเบราว์เซอร์

3) อัปขึ้นโฮสต์แบบ Static
   - เลือกบริการโฮสต์ static site ที่คุณสะดวก (แบบ drag-and-drop หรือฟีเจอร์ Pages ของแพลตฟอร์มโค้ดโฮสติง)
   - อัปโหลดไฟล์ทั้งหมดนี้ขึ้นไป แล้วคุณจะได้ URL สาธารณะ

ทำ QR CODE
----------
ตัวเลือก A: ใช้เว็บสร้าง QR ใดๆ
  - ใส่ URL เว็บที่โฮสต์แล้ว แล้วดาวน์โหลดเป็น PNG

ตัวเลือก B: ใช้ Python ในเครื่องคุณ
  - ติดตั้งไลบรารี:  pip install qrcode[pil]
  - รันสคริปต์   :  python qr.py "https://your-domain.example/menu"
  - จะได้ไฟล์ qr.png พร้อมพิมพ์

เคล็ดลับภาพ
-----------
- พื้นหลังโปร่งใสใช้ PNG, รูปถ่ายใช้ JPG
- ถ้ารูปใหญ่มาก ให้บีบอัดก่อนเพื่อความเร็ว
- ตั้งค่า alt ใน index.html ให้เหมาะกับเนื้อหา
